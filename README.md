# AfterHours Nightlife Website

A modern web application built for discovering and experiencing nightlife.

## Tech Stack

- **Frontend:** React 18, Vite, Tailwind CSS, Framer Motion, Lucide React
- **Backend:** Node.js, Express
- **Routing:** React Router DOM

## Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

**Development Mode:**

```bash
npm run dev
```

**Production Build:**

```bash
npm run build
```

**Preview Production Build:**

```bash
npm run preview
```

**Start Server:**

```bash
npm start
```

## Project Structure

- `src/`: Contains the React frontend code (components, pages, etc.)
- `public/`: Static assets
- `server.js`: Express backend entry point
- `vite.config.js`: Vite configuration

- **Web Animations:** Bringing modern web aesthetics to life by implementing smooth, declarative animations and transitions using **Framer Motion**.
